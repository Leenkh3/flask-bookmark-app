A Flask-based web application for managing bookmarks.

## Features
- Add, edit, and delete bookmarks.
- Auto-fetch metadata (title and description) for URLs.
- Tag-based filtering and sorting.

## Requirements
- Python 3.8 or higher
- Flask
- SQLite
